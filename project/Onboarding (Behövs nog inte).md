# Onboarding Flow
Denna behövs nog inte, vi kanske dock kan ha med en välkomstskärm med någon snygg animation om vi har tid
## Välkomstskärm
- [ ] Välkomsttext med appförklaring
- [ ] "Vad är Migränappen?" kort beskrivning
- [ ] Illustration/animation av huvudfunktioner
- [ ] "Kom igång" knapp
## Check-in Frekvens
- [ ] "Hur ofta vill du få påminnelser att checka in?"
- [ ] Val: 2 gånger per dag / 3 gånger per dag / Anpassad
- [ ] Tidpunktsväljare för varje check-in
- [ ] Preview av hur notiser kommer se ut
## Behörigheter
- [ ] Notifikationer (med förklaring varför)
- [ ] Hälsodata (valfritt)
- [ ] Plats (för väderdata - valfritt)
- [ ] "Hoppa över" alternativ för allt
## Snabbguide
- [ ] Interaktiv tutorial med tooltips
- [ ] "Så här trackar du" (swipe-through)
- [ ] "Så här fungerar AI-varningar"
- [ ] "Hoppa över tutorial" knapp
## Första Profilen
- [ ] "Lite om dig" (valfritt)
- [ ] Ålder, kön (för bättre mönster)
- [ ] "Hur länge har du haft migrän?" slider
- [ ] "Hoppa över" möjlighet
## Klart!
- [ ] "Du är redo!" bekräftelse
- [ ] Snabb sammanfattning av vad som händer nu
- [ ] "Börja använda appen" knapp
