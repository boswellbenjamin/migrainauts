# Logga Migrän

## Snabblogg (Från FAB-knapp)
### Steg 1: Bekräfta
- [ ] "Har du migrän just nu?" stor text
- [ ] "Ja" (stor röd knapp)
- [ ] "Nej, men jag kände symptom" (mindre knapp)
- [ ] "Avbryt"

### Steg 2: Tidsstämpel
- [ ] Förvald: "Just nu" (current time)
- [ ] Justera tid bakåt (om migrän började tidigare)
- [ ] Datum och klockslag picker
- [ ] "Nästa" knapp

### Steg 3: Intensitet
- [ ] Stor slider (1-10)
- [ ] Visuell feedback (färg ändras, emoji)
- [ ] Labels: "Mild", "Måttlig", "Svår", "Invalidiserande"
- [ ] "Nästa" knapp

  - Pulserande smärta
  - Aura
  - Ensidig smärta
- [ ] "Lägg till fler symptom" länk
